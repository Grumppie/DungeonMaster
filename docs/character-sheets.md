## Demo assumptions

- Single duel arena with no cover and a starting distance of 20 feet.
- The app runs a strict 2014 5e-style combat loop for initiative, attacks, saves, damage, rage, resistance, concentration, opportunity attacks, and death saves.
- Session state is cloned from `data/initial-state.json` into browser storage and reset after the duel ends.
